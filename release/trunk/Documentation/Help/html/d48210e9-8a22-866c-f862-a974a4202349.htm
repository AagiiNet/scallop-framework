IScallopNetwork Events

The IScallopNetwork type exposes the following events.

Events

 Public

 Protected

 MemberDescription
Data
This event is raised when new network data is received. The received message is passed in the event arguments.
Info
This event is raised when the network module wants to inform the user of something interesting. The message is passed in the event arguments.
StatusChanged
This event is raised when there is an error with the network. The reason for the error is contained in the event arguments.
